{% trans "Начало работы" %}

{% trans "Как зарегистрироваться?" %}▾
{% trans "Перейдите на страницу регистрации, выберите роль (Покупатель/Поставщик), укажите ИНН компании. После подтверждения email вы получите доступ к кабинету. Полная активация — после прохождения KYB верификации (1-2 рабочих дня)." %}
{% trans "Что такое KYB и зачем он нужен?" %}▾
{% trans "KYB (Know Your Business) — проверка легальности компании. Загрузите Устав, Выписку ЕГРЮЛ и паспорт директора в разделе /kyb/. Без верификации недоступны крупные сделки и эскроу." %}
{% trans "Можно ли работать командой?" %}▾
{% trans "Да. В разделе «Команда» пригласите коллег по email с ролью (Админ/Менеджер/Логист/Финансист/Только просмотр). Один аккаунт компании — до 50 пользователей." %}

{% trans "Заказы и RFQ" %}

{% trans "Что такое RFQ?" %}▾
{% trans "RFQ (Request For Quote) — запрос котировки. Покупатель указывает нужные запчасти, поставщики отвечают ценами и сроками. Сравните предложения и выберите лучшее." %}
{% trans "Как отследить заказ?" %}▾
{% trans "В разделе «Мои заказы» каждый заказ имеет статусы: Резерв оплачен → В производстве → Транзит → Таможня → Доставлен. SLA контроль показывает дедлайны." %}
{% trans "Что делать при проблемах с заказом?" %}▾
{% trans "Откройте рекламацию в разделе «Рекламации» — приложите фото повреждений или несоответствий. Оператор рассмотрит в течение 14 дней." %}

{% trans "Оплата" %}

{% trans "Как устроена оплата?" %}▾
{% trans "Трёхступенчатая схема: 10% резерв (подтверждение заказа) → 50% основной платёж (готовность к отгрузке) → 40% таможенный платёж (после таможни). Все средства проходят через эскроу оператора." %}
{% trans "Можно ли вернуть резерв?" %}▾
{% trans "Да, до момента отправки в производство — резерв возвращается без штрафа. После — взимается 10% штраф за отмену." %}

{% trans "Логистика" %}

{% trans "Какие способы доставки доступны?" %}▾
{% trans "Море (контейнер 20/40ft), авиа (экспресс/стандарт), ж/д. Маршруты: Китай → Москва, Тайвань → Москва, Япония → Москва. Калькулятор в разделе «Логистика»." %}
{% trans "Кто оформляет таможню?" %}▾
{% trans "Оператор. Вы загружаете инвойс и упаковочный лист, брокер платформы готовит декларацию и оплачивает пошлины." %}
