API для интеграций
REST API для получения отчётов, метрик и уведомлений в ваших системах
API ключ
sk-cons-4f8a2b1c9d7e6f3a5b0c8d2e1f4a7b3c
Используйте этот ключ в заголовке Authorization: Bearer <key>
Эндпоинты
GET/api/v1/reports/sales
Отчёт по продажам и выручке. Параметры: period, format (json/csv/pdf)
GET/api/v1/reports/sla
SLA метрики и нарушения. Параметры: period, include_details
GET/api/v1/reports/conversion
Воронка конверсии RFQ → Заказ. Параметры: period, group_by
GET/api/v1/reports/finance
Финансовый отчёт P&L. Параметры: period, currency, include_debtors
GET/api/v1/reports/rating
Рейтинг и история событий. Параметры: period, include_events
GET/api/v1/reports/logistics
Логистика и отгрузки. Параметры: period, destination_country
GET/api/v1/metrics/dashboard
Сводка ключевых метрик дашборда в реальном времени
POST/api/v1/reports/generate
Генерация пакетного отчёта. Body: { reports: [...], period, format, style }
POST/api/v1/webhooks/subscribe
Подписка на webhook уведомления. Body: { url, events: [...] }
POST/api/v1/notifications/send
Отправка отчёта в Telegram/WhatsApp. Body: { channel, report_id, chat_id }
Документация API доступна по адресу docs.consolidator.parts/api/v1. Rate limit: 100 запросов/минуту. Формат ответа: JSON. Поддерживается пагинация, фильтрация и сортировка.
